# Игра 2048 была создана на УП для колледжа ГБПОУ КТК

## Технологии

- Язык: **Java 17**
- Android:
  - `compileSdk`: 36
  - `targetSdk`: 36
  - `minSdk`: 26
- Сборка: **Gradle Kotlin DSL**
- UI:
  - `ConstraintLayout`
  - `GridLayout` для игрового поля
  - компоненты Material (карточки score/best)

## Архитектура и слои

Код разделён на три основных уровня:

- **UI слой** — MainActivity, отвечает за:
  - отрисовку поля;
  - обработку свайпов и нажатий стрелок;
  - управление состоянием игры;
  - вызовы бизнес-логики.

- **Доменная модель** — Field, Position, State, Cache.
  - Простые структуры данных, не содержащие логики.

- **Доменные сервисы** — FieldService, MoveService, CacheService.
  - чистая бизнес-логика:
    - генерация новых плиток;
    - поиск свободных ячеек;
    - повороты матрицы;
    - движение и сжатие плиток;
    - подсчёт очков;
    - кэширование undo.

## Обработка ввода

- **Свайпы**:
  - GestureDetector отслеживает onFling().
  - Направления мапятся на команды State.
  - Вызывается rotateField(), затем move(), затем отрисовка.

- **Клавиатура**:
  - onKeyDown() принимает KEYCODE_DPAD_LEFT/RIGHT/UP/DOWN.
  - Маппинг идентичен жестам.

## Геймплей

- Поле 4×4.
- Плитки 2 и 4 генерируются случайно.
- После каждого успешного хода создаётся новая плитка.
- Реализована поддержка:
  - отмены хода (undo),
  - счёта,
  - лучшего результата (best score),
  - автоматического рестарта при отсутствии ходов.

## Интерфейс

- Сетка 4×4 через GridLayout.
- Карточки счёта через MaterialCardView.
- Адаптивная верстка с ConstraintLayout.
